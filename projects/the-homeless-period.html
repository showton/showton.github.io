THE HOMELESS PERIOD STYLE GUIDE

the prompt of this final project was to redesign and rebrand a non-profit organization by creating a style guide that would outline which fonts, colors, logos, graphic elements, and photography should be used.


PROJECT INTENT

For the final project, I chose The Homeless Period, an organization dedicated to giving free feminine hygiene products to homeless women. The style guide includes:

a mood board
logo usage
font choices
a weighted color palette
graphic elements
photography the brand should use
web design mockups
package design

Created using Adobe Illustrator, Photoshop, and InDesign
